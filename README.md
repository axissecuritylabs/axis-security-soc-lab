## What Is This?

SOC Lab is a hands-on simulation environment designed to bridge the gap between cybersecurity certifications and real-world SOC work. Built by Axis Security Solutions, it puts you through the same decision-making process a working analyst faces every day — triaging alerts, investigating enrichment data, making escalation decisions, and documenting your work.

Everything runs in your browser using localStorage. No data leaves your device.

---

## What's Inside

**76+ Realistic Alerts** across Tier 1 and Tier 2 categories including phishing, credential attacks, malware execution, lateral movement, ransomware, insider threats, cloud misconfigurations, and more.

**Alert Queue** — Claim tickets, enrich fields, make triage decisions, and receive coaching on every outcome.

**Interactive Investigation Checklist** — Task-gated steps that require written answers before you can advance. Every step reveals why it matters.

**Incident Response Console** — Escalated alerts funnel into a full NIST SP 800-61 IR lifecycle simulation: Containment → Eradication → Recovery → Lessons Learned. Execute containment actions, block indicators, reset credentials, isolate hosts, and reverse any action. Formal deconfliction notes required for closure.

**Detection Engineering Lab** — Three fully interactive guided demos (macro/PowerShell, DNS C2 beaconing, persistence via scheduled task) with multiple choice, checkbox exercises, TP/TN/FP classification, and write-your-own Sigma/SPL rule exercises before the reference answer is revealed.

**Threat Hunt Lab** — Five hunt categories with hypothesis-driven exercises and validation hints.

**Forensics Lab** — YARA rule writing, memory forensics, log analysis, cloud forensics, and automation exercises.

**File Viewer** — Inspect intercepted files inside DLP and malware alerts. Spreadsheets with PII flagged
